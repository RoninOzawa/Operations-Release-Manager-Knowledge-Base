# 1 生命周期概述

---
- 生命周期就是用户在使用某一资源位、模块、功能时，动作随时间推移的完整变化过程。不同目标场景下，观测链条和核心指标各有不同。即：
```
生命周期 = 时间轴上的用户行为变化
场景不同 = 观测链条不同

一套万能的生命周期阶段划分：

触达期 → 激活期 → 成长期 → 稳定期 → 衰退期 → 流失期 → 召回期
```
## 1.1 常用关键指标

```
激活：启动率、首局完成率
成长：D1/D7 留存、复玩率、使用时长
稳定：DAU、WAU、MAU、会话频次
衰退：活跃下降率、流失率
召回：回流率、召回转化率
商业化：付费率、ARPU、LTV
```

# 2 生命周期划分
- 生命周期有通用框架，但没有唯一标准答案。核心阶段通常相似，具体怎么分，取决于要解决什么问题。
- 生命周期全阶段划分的细致程度，被称为**阶段粒度**。粒度越粗越适合看整体，但不容易定位问题。粒度越细致越容易复盘找问题，但埋点和分析成本更高。

## 2.1 按业务目标划分
```
最常用的分析方法，团队关心什么，就观测什么：
获客生命周期：曝光 → 点击 → 到达 → 激活
留存生命周期：激活 → 复玩 → 稳定使用 → 流失
商业化生命周期：使用 → 付费 → 复购 → 召回
```
## 2.2 按用户关系划分
```
按用户关系划分，不是看用户做了什么动作本身，而是看用户和平台当前处于什么关系状态，然后决定给他什么内容、什么资源位、什么运营动作。
我们常说的用户生命周期，一般就是指用户与平台关系周期。

用户与平台关系生命周期：陌生用户 → 新用户 → 活跃用户 → 高价值用户 → 流失用户 → 召回用户

我们再拆解一下每个阶段的运营内容
1）潜在用户运营
	参考指标：曝光、点击、到达、首激活
	运营内容：资源位优化、内容优化、页面承接优化、首次转化提升
	运营目的：平台拉新

2）新用户运营
	参考指标：新注册、新首次启动、新首次访问
	运营内容：新手引导、首次激活、首屏推荐、资源位首推、新手任务设计
	运营目的：保证新用户留存

3）活跃用户运营
	参考指标：日活、周活、月活
	运营内容：日常内容推荐、高频资源位排期、习惯培养、稳定曝光、栏目运营
	运营目的：培养用户习惯，稳定平台活跃度，维持平台流量，尝试将其朝高价值用户转化

4）高价值用户运营
	参考指标：使用时长、使用频次、玩家标签
	运营内容：细分活动、高价值内容优先推荐、定制化资源位、召回和维护、付费转化
	运营目的：培养用户习惯，做差异化，高价值用户更有可能向付费用户转化。

5）流失用户运营
	参考指标：回流率、召回率、召回后次日留存
	运营内容：Push 召回、限时福利、回流专题、重新激活、低门槛入口推荐
	运营目的：尝试召回长期流失的用户，维持平台流量

6）已召回用户运营
	参考指标：复玩率、7日留存、再次流失率
	运营内容：回流承接页、老用户专属活动、复玩激励、使用习惯恢复
	运营目的：稳定已召回用户，尝试将其朝活跃用户转化

7）付费用户运营
	参考指标：付费、首充、复购、重付费
	运营内容：付费漏斗优化、商品推荐、会员/礼包/折扣、商业化资源位
	运营目的：高价值用户变现

```
## 2.3 按产品类型划分
```
不同的产品类型，其功能周期也有所不同

游戏：启动、游玩、复玩、付费
内容平台：浏览、关注、互动、回访、付费
工具产品：安装、激活、持续使用、订阅、流失
```
